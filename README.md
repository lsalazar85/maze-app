# Maze game app
Maze game developed with Next.js

## Run Project
Starts on `http://localhost:3000`

### Npm
```
    npm install
    npm run dev
```

### Yarn
```
    yarn install
    yarn run dev
```

## Development Technologies
* Modern Javascript ( ES6 )
* Next.js
* React.js
* React Testing Library & Jest
* Styled-Components
